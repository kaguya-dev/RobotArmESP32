# RobotArmESP32
RobotArm oriented for Inverse Kinematics movement and Bluetooth connection from the ESP32 Board.


# Components
## Materials
- 4 Servos MG996R
- Power Supply 12v 10A
- Esp32-32D (30Pin version)
- 3d Filaments for material
- 4 1000uF 25v Capacitors (For Servo)
- A Home made pcb for creating the Power Distribuition Panel (Will be called in this project PDP)

## Code
We are using the Arduino framework for more family friend testing and development, recomended for anyone trying to redo the project.
